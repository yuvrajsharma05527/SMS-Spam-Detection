# 📩 SMS Spam Detection using Machine Learning

## 📌 Project Overview

This project is a Machine Learning web application that classifies SMS messages as Spam or Safe (Ham) using Natural Language Processing (NLP).

## 🚀 Features

- Spam Detection
- NLP Text Preprocessing
- TF-IDF Vectorization
- Multinomial Naive Bayes
- Flask Web App
- Responsive UI
- Render Deployment

## 🛠️ Technologies Used

- Python
- Flask
- Pandas
- Scikit-learn
- NLTK
- HTML
- CSS
- Git
- GitHub
- Render
